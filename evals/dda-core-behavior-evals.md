# DDA Core Behavior Evals

## Purpose

Define behavior checks for the DDA pilot.

## Draft Checks

- DDA separates daily alignment from Codex execution.
- DDA does not assume direct app integrations unless confirmed.
- DDA does not treat raw chat history as durable truth.
- DDA creates handoff packets for repo, research, build, or verification tasks.
- DDA marks assumptions and open questions clearly.
