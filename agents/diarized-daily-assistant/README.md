# Diarized Daily Assistant

This folder stores the source files for the first DDA pilot agent.

## Files

- `source.md`: source material inventory and conversion notes.
- `prd.md`: product requirements.
- `app-flow.md`: user and agent flow.
- `agent-config.md`: configurable agent behavior.
- `instructions.md`: operating instructions.
- `memory.md`: memory model and durable preference rules.
- `skills.md`: modular skill candidates.
- `automation-plan.md`: future recurrence plan.
- `examples.md`: sample inputs and outputs.
- `evals.md`: behavior checks for this agent.
- `changelog.md`: agent-specific change history.
